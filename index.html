SCORM - API functions

The API consists of the following functions:

  • LMSInitialize("")
  • LMSGetValue("name")
  • LMSSetValue("name", "value")
  • LMSCommit("")
  • LMSFinish("")
  • LMSGetLastError()
  • LMSGetErrorString("errorcode")
  • LMSGetDiagnostic ("errorcode")
